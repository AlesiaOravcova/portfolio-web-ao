Your home workout

Personalised fitness app to motivate people to create exercise routine and become fit and healthy.

UI Case Study

It was the course project of UI for UX designers, that helped me to practice UI principles and techniques.

My role

UI Designer

Tools

Figma, FigJam

Duration

3 months

Objective

To create a personalised fitness web app for busy people that motivates into an exercise routine and suits their level, schedule, and interests.

Context & Challenge

Exercising at home is an increasingly popular way how to be fit and healthy. Building exercise routine could be challenging in several aspects:

  • to be motivated to exercise regularly,
  • to fit workout into the busy schedule,
  • to find exercise for your level.

Therefore, this app is aimed to encourage people to get into an easy exercise routing by

  • providing guides and interactive examples,
  • fitting as little as 5 minutes routine into their calendars, and
  • setting their goals and tracking the results.

User Flow

Considering User Persona's goals and needs, and based on the User Stories provided in the Project Brief, I have created User Flows to visualise the process how the users can reach their goals.
User Flow: Choose Exercise
User Story 1:

As a new user, I want to learn about different exercises, so that I can figure out what is best for me.

User Story 2:

As a new user, I want to be shown how the exercises are done, so that I know I’m doing them correctly.

Low Fidelity Wireframes

Sketching the User Flows on the paper helped me to gather the thoughts and to conceptualise functions and app navigation.

User Flow: Choose Exersice

Mid Fidelity Wireframes

Working with mid fidelity wireframes in Figma, I've structured the screens, placed UI elements, explored design patterns following the grid, spacing and hierarchy moving into more and more detailed design.
